
# Webpack template


Clone the project

```bash
  git clone https://github.com/nereid-n/webpack-template.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Build the project

```bash
  npm run build
```

Create new page

```bash
  npm run create-page
```

Create new component

```bash
  npm run create-component
```
